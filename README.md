A research-focused project to practice Machine Learning concepts. The original dataset was actually too clean, so I had to intentionally introduce noise to simulate real-world scenarios!
- **Goal:** Classify water quality using real-world data simulations.
- **What I learned:** Handling noisy data, feature engineering, and comparing the accuracy of models like **Random Forest** and **XGBoost** (and distinguishing them from simpler models like standalone Decision Trees).
